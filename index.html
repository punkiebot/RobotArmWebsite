Introducción

Brazo robótico controlado por PC, con 6 grados de movimiento, se controla a través de una interfaz grafica desarrollada con JavaScript, HTML y CSS, esta GUI tiene varios botones para ejecutar rutinas ya programadas para cada servomotor, una rutina para todos los motores y un botón para mover el brazo a su posición por defecto, también tienes sliders (deslizadores) para seleccionar el ángulo de giro de cada robot.

El control de los servomotores lo hace la placa Raspberry Pi Pico usando el lenguaje MicroPython, esta se comunica por protocolo UART con el computador usando el módulo conversor USB A Serial RS232 UART TTL CP2102.

En el computador hay un programa en Python que se comunica con la placa Raspberry Pi usando la biblioteca PySerial, este programa Python continuamente lee un archivo JSON y cuando detecta cambios los envía a la placa.

Los cambios efectuados en el archivo JSON los hace un programa del Backend que corre en Node, este programa recibe los comandos de la interfaz grafica y los guarda en el archivo JSON

Descripción

Interfaz Gráfica

6 barras deslizadoras, una para cada servomotor, 8 botones, 6 de los cuales ejecutan rutinas para cada motor, el séptimo una rutina para todos los motores y el último botón lleva el brazo a su posición por defecto.

La imagen de la mitad es una guía para identificar cada servomotor.

Fuente de poder

Se trabajó con un adaptador de voltaje 12v y 5 amperios, junto con un reductor de voltaje a 6v para alimentar los servomotores.
